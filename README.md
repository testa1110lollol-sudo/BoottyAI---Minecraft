# 🤖 BottyAI — Автономный Minecraft Бот (APLHA)

Умный бот для Minecraft, который самостоятельно выживает, собирает ресурсы, крафтит инструменты и исследует мир. Построен на [Mineflayer](https://github.com/PrismarineJS/mineflayer).

## ✨ Возможности

### 🧠 Автономное поведение
- **Система целей** — бот выполняет одну задачу за раз и переключается на следующую
- **Прогрессия по эпохам** — Wood Age → Stone Age → Iron Age → Diamond Age
- **Антизависание** — детекция застревания и автоматическое освобождение
- **Таймауты** — каждый тип задачи имеет свой лимит времени

### ⚔️ Выживание
- **Самозащита** — атакует враждебных мобов в радиусе 12 блоков
- **Побег** — убегает при низком здоровье (< 6 HP)
- **Умная еда** — ест когда голоден, охотится только при критической нехватке
- **Плавание** — держится на плаву и ищет берег

### 🪵 Сбор ресурсов
- Добыча дерева (все типы брёвен)
- Добыча камня и руды
- Сбор урожая (пшеница, морковь, картофель)
- Лут сундуков в деревнях

### 🔨 Крафт
- Доски, палки
- Верстак
- Деревянная и каменная кирка
- Автоматическая установка верстака

### 🗺️ Исследование
- Случайное блуждание с избеганием опасностей
- Поиск деревень по признакам (кровати, дорожки, жители)
- Отслеживание игроков

## 📋 Требования

- **Node.js** 16.0 или выше
- **npm** или **yarn**
- **Minecraft сервер** (Java Edition) версии 1.8 - 1.20
